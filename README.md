# project name : MovieManager
Movie App that :
1- @GET POPULAR,TRENDING, and UPCOMING movies from MOVIEDB api using Retrofit2.0 library                
2- Login and SignUp Authentic. Activities Using FirebaseAuth.              
3- Menu Option of SignOut Click.

# Technology used:
Java
